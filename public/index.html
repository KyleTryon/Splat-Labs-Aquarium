Splat Labs Aquarium

A TypeScript and WebGL browser aquarium simulator, inspired by early 2000's webgames like Nick-O-Quarium, and Insaniquarium You play a mad scientist creating new and unique aquatic life from "splat" a 90's - 2000's slime like substance.

What is going on here?

Hello! My name is Kyle, a Youtuber/Programmer/DevOps Engineer. This is a live updating project I am currently working on as of AUG 2020.

The above is a WebGL powered game/simulation using TypeScript and the Phaser 3 framework. Each fish operates on a basic level of "AI" based on routines, and priority. Each fish has a set of traits, stats, and needs which govern the priority of each routine. Currently fish only whave 2 routines, "Wander" and "Hunt". Click the screen to add food.
